## Overview

GH-OPS Guide は、**企業・組織で GitHub を安心して・持続可能に運用するための実務ガイド**です。  
公式ドキュメントを補完し、現場の判断を速く安全にするための「共通言語」を提供します。

👉 詳細な構成・目次は [Index](src/content/docs/index.mdx) をご覧ください。
